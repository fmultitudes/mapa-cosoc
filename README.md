# Mapa COSOC
Mapa interactivo de Chile, parte del sitio del reporte de COSOC.
Información obtenida desde un Google Spreadsheet.

## Desarrollo

`npm install`

`bower install`

`gulp`

Servidor local: `http://localhost:10000` - Live reload incluído. 

## Tecnologías
- Lodash
- D3.js
- Tabletop
- Knockout.js
- TopoJSON
